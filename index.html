    About Us

    This portal is a prototype of a Large Language Model powered assistant that helps citizens explore eligibility for government schemes in multiple languages.

    The frontend uses an accessible glassmorphism UI built with Tailwind CSS, while the backend (to be integrated) will use Python, LangChain, Gemini API, and a vector database to match user profiles with schemes.
